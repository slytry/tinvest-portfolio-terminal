# 📥 Скачивание и конвертация облигаций с dohod.ru

Скрипты для автоматического скачивания данных об облигациях с сайта [dohod.ru/analytic/bonds](https://www.dohod.ru/analytic/bonds) и преобразования в JSON с датой в имени файла.

## 🚀 Возможности

- 🤖 Автоматическое скачивание Excel-файла
- 📅 Добавление даты и времени в имя файла (`bonds_2026-03-06_23-41.json`)
- 🔄 Конвертация в структурированный JSON для веба
- 📊 Расчет базовой статистики

## 📋 Требования

- Node.js 14+
- npm

## 🔧 Установка

```bash
# Создать папку проекта
mkdir bonds-downloader
cd bonds-downloader

# Инициализировать проект
npm init -y

# Установить зависимости
npm install puppeteer xlsx date-fns
